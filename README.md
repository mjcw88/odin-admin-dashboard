# The Odin Project (Full Stack Javascript) - Admin Dashboard

Admin Dashboard project for the Odin Project Full Stack Javascript course.

Main goal of this project is to create an entire layout solely with css grids rather than flexboxes.

Full project specs can be found [here](https://www.theodinproject.com/lessons/node-path-intermediate-html-and-css-admin-dashboard).
